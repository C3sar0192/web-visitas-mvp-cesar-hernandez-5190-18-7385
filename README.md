Este repositorio contiene el codigo fuente WEB del proyecto de visitas técnicas para el seminario de privados de la sede de antigua. 

COMO EJEUTAR EL PROYECTO WEB EN LOCAL

Este proyecto es una aplicación **Web MVC en .NET 8** que consume una **API REST** de visitas.  
Sigue estos pasos para levantarlo en tu máquina.

---

##CLONAR EL REPOSITORIO

Abre una terminal (CMD, PowerShell o Git Bash) y ejecuta:

bash
git clone https://github.com/C3sar0192/web-visitas-mvp-cesar-hernandez-5190-18-7385.git
(Si no se quiere clonar solo descarga el proyecto)

##EJECUTAR EL PROYECTO
Dirigirse a raiz del proyecto en el Web con comando "cd" desde una terminal CMD, PowerShell
Ejemplo: cd E:\Proyectos\web-visitas-mvp-cesar-hernandez-5190-18-7385\Web
Dentro de la carpeta Web ejeutar los siguientes comandos:

ejecutar "dotnet restore" ---- Esto descargará todas las dependencias necesarias para compilar y ejecutar el proyecto.
ejecutar "dotnet build"
ejecutar "dotnet run" esto para correr el proyecto
